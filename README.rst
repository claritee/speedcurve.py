speedcurve
=============

A wrapper for around the `SpeedCurve API v1`_

`Full documentation`_

Example Use
-----------

.. code-block:: python

   import speedcurve

   sc = speedcurve(api_key='your-api-key-here')
   sites = sc.sites()


.. _SpeedCurve API v1 : https://api.speedcurve.com/
.. _Full documentation: http://speedcurvepy.readthedocs.org/
